# 🌐 Personal Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript. It serves as an online platform to showcase my skills, projects, and information as a web developer.

## 🚀 Features

* Clean and modern UI design
* Responsive layout for different screen sizes
* Sections for:

  * Home
  * About Me
  * Projects
  * Contact
* Smooth navigation
* Organized and readable structure

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```id="pers123"
├── index.html
├── style.css
├── script.js
├── assets/ (images, icons, etc.)
```

## 🎯 Project Goal

The main goal of this project is to create a personal online presence and showcase my work as a developer. Personal portfolio websites are important because they act as a **digital resume and project showcase** for developers ([KaaShiv InfoTech][1]).

## ⚙️ How It Works

* Displays personal information and skills
* Showcases projects with descriptions
* Allows users to navigate easily between sections
* Can be used by recruiters to view my work

## ▶️ How to Run

1. Clone the repository:

   ```
   git clone https://github.com/rinshuDeveloper/personal_website.git
   ```

2. Open the project folder

3. Run `index.html` in your browser
